**StoryFrame** — це інтерактивний IoT-пристрій для картин та фоторамок. Коли людина наближається до картини, пристрій автоматично розпізнає це за допомогою високоточного лазерного далекоміра (Time-of-Flight) та починає відтворювати аудіоісторію або музику з SD-карти у високій якості через шину I2S.

Пристрій має стильну веб-панель керування (SPA) для налаштування, завантаження MP3-треків по Wi-Fi та вибору режимів відтворення.

## ✨ Основні можливості

- **Миттєва реакція:** Лазерний датчик VL53L0X безпомилково фіксує наближення людини на заданій відстані (наприклад, 20 см).
- **Чистий цифровий звук:** Декодування MP3 та вивід аудіо через I2S ЦАП (MAX98357A) без фонових шумів.
- **Веб-адмінка (LittleFS):** Локальний веб-інтерфейс з сучасним дизайном для керування пристроєм.
- **Керування треками:** Завантаження `.mp3` файлів безпосередньо на SD-карту через браузер.
- **Smart Boot (Captive Portal):** Автоматичне створення точки доступу `StoryFrame_Setup` при першому запуску або втраті Wi-Fi.
- **Режими відтворення:** Одинарний трек або режим "Плейлист" (перемикання треків при кожному новому наближенні).
- **OTA Оновлення:** Підтримка оновлення прошивки "по повітрю".

## 🛠 Апаратна частина (BOM)

Для збірки проєкту вам знадобляться:

1. **Мікроконтролер:** ESP32-S3 DevKitC-1 (N16R8 — 16MB Flash, 8MB PSRAM) + зовнішня антена IPEX для стабільного Wi-Fi.
2. **Аудіо-підсилювач:** MAX98357A (I2S Class D Amplifier) + Динамік 4 Ом, 3 Вт.
3. **Датчик наближення:** Модуль TOF200C (на базі чіпа VL53L0X, I2C).
4. **Сховище:** MicroSD Card Module (SPI, версія на 3.3V без конвертера рівнів) + MicroSD карта (FAT32).
5. **Живлення:** Блок живлення 5V / 2A та електролітичний конденсатор 1000 мкФ (10-16V).

## 🚀 Встановлення та прошивка

Проєкт створено в середовищі розробки **PlatformIO**.

1. Клонуйте репозиторій:
   ```bash
   git clone [https://github.com/YeKhomenko/StoryFrame.git](https://github.com/YeKhomenko/StoryFrame.git)
   cd StoryFrame
Відкрийте папку проєкту у VS Code з встановленим розширенням PlatformIO.

Прошивка Файлової Системи (Обов'язково!):

Перейдіть у меню PlatformIO на бічній панелі.

Відкрийте Project Tasks -> env:esp32-s3-devkitc-1 -> Platform.

Натисніть Build Filesystem Image, а потім Upload Filesystem Image, щоб завантажити веб-інтерфейс з папки data у пам'ять ESP32 (LittleFS).

Прошивка Коду:

Натисніть кнопку Upload на нижній панелі VS Code, щоб скомпілювати та залити основний C++ код (main.cpp).

⚙️ Використання (Перший запуск)
Увімкніть пристрій.

Знайдіть на смартфоні чи ноутбуці Wi-Fi мережу StoryFrame_Setup та підключіться до неї (пароль: 12345678).

Відкрийте браузер і перейдіть за адресою http://192.168.4.1.

Авторизуйтесь зі стандартними даними (admin / admin).

У веб-адмінці введіть дані вашої домашньої Wi-Fi мережі та збережіть. Пристрій перезавантажиться і підключиться до вашого роутера.

Завантажте MP3-файли у вкладці "Файли" і насолоджуйтесь магією!
